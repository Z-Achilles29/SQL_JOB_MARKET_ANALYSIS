# SQL_JOB_MARKET_ANALYSIS
Analyzed the highest-paying Data Analyst jobs using SQL to uncover salary trends, in-demand skills, top companies, and geographic insights. Demonstrates SQL querying, data exploration, and business intelligence techniques.

📊 Top 10 Highest-Paying Data Analyst Jobs by Skill (SQL Project)
📌 Project Overview

This SQL project analyzes the highest-paying Data Analyst jobs and the technical skills required to secure these roles. Using SQL queries, the dataset was explored to identify salary trends, in-demand skills, top-paying companies, and geographic distribution of jobs.

The objective is to help aspiring Data Analysts understand which skills are most valuable for landing high-paying opportunities and make informed learning decisions.

🎯 Objectives
Analyze the highest-paying Data Analyst jobs.
Identify the most frequently required technical skills.
Discover salary trends across companies and countries.
Find which skills appear most often in high-paying positions.
Generate insights using SQL queries.
🛠️ Tools & Technologies
SQL
PostgreSQL / MySQL (depending on your database)
Git & GitHub
📂 Dataset Information

The dataset contains information about Data Analyst job postings, including:

Job ID
